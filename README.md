# Sistema de Gestão de Agendamentos

## Status do Projeto
Em desenvolvimento

## Tecnologias
- Java
- MySQL
- JDBC (para conexão com o banco de dados)

## Desenvolvedor
- Juan Gabriel

## Objetivo do Software
Este software tem como objetivo gerenciar de forma prática clientes, serviços, profissionais e agendamentos em um ambiente desktop, facilitando o controle de informações e operações de um pequeno negócio.

## Funcionalidades do Sistema
- Cadastro de clientes
- Cadastro de profissionais
- Cadastro de serviços
- Registro de agendamentos
- Consulta e edição de dados
- Relatórios básicos de agendamentos e clientes
